## Project context
ProtoCall Trainer ("CrewTable") — a live tactical fireground & EMS scenario simulation hub.
Host generates a live session → crew scans QR to join → crew submits answers →
host pushes notable answers to all devices → session archives to PDF/account.

See SPEC.md for the full technical specification and data schemas.
The interactive prototype is ProtoCall_trainer.html (single-file, Tailwind + Lucide).
